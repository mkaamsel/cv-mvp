# Prompt Principles

## Non-negotiable rules
- use only information present in the CV or profile
- never invent experience, tools, languages, leadership scope, industries, or achievements
- prefer omission over fabrication
- preserve credibility over style
- tailor by emphasis, not by invention

## Output rules
- keep outputs compact and readable
- prioritize relevance to the target role
- reduce repetition
- preserve professional tone
- match output language cleanly
- adapt writing level without changing facts

## Role-fit rules
- warning should appear when mismatch is meaningful
- weak fit should be communicated gently and clearly
- summary should explain strengths and gaps, not only score

## Quality rules
- avoid generic filler phrases
- avoid exaggerated praise
- avoid inflated leadership language
- avoid unnatural German formulations
