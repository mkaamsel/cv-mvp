# Rollout Plan

## Stage 1
Founder use and close associates.

Goals:
- detect failures
- improve output quality
- validate role-fit warnings
- polish UI

## Stage 2
Extended trusted testers.

Goals:
- collect structured ratings
- validate stability
- test multiple profiles and industries
- identify onboarding friction

## Stage 3
Public controlled rollout.

Channels:
- LinkedIn
- personal referrals
- selected communities

Goals:
- validate demand
- test pricing response
- observe market differences between Germany and India

## Rollout blockers
Do not scale rollout if:
- outputs still invent facts
- warning logic is unreliable
- DOCX parsing is unstable
- feedback capture is broken
- UI confuses first-time users
