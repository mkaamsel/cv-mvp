# Canonical Profile Schema

## Purpose
The canonical profile is the stable structured truth about a candidate, derived from CV and user-confirmed edits.

## Fields
- headline
- professional_summary
- highest_education
- years_experience_estimate
- seniority_level
- leadership_experience
- target_fields
- target_roles
- non_target_roles
- languages
- core_skills
- tools_and_erp
- reporting_frameworks
- industries
- certifications
- location
- work_authorization_if_collected
- facts_never_to_invent
- notes_for_tailoring

## Principles
- generated once, then editable
- reused across tailoring sessions
- reduces repeated parsing
- improves consistency
- lowers model cost
