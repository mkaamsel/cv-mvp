# Risk and Guardrails

## Main risks
- hallucinated qualifications
- weak-fit false positives or false negatives
- poor DOCX extraction quality
- generic or repetitive cover letters
- unnatural German phrasing
- overconfident tone
- emotionally insensitive wording
- excessive OpenAI cost from repeated parsing

## Guardrails
- canonical profile must become the stable truth layer
- source CV text must be preserved
- prompts must forbid invention
- warnings must be soft but clear
- input limits should remain enforced
- outputs should be reviewable and rateable

## UX guardrails
- errors should be clear and polite
- warning banners should not feel punitive
- user should always understand what happened
