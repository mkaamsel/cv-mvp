# Feedback Learning

## Why feedback matters
Feedback is the fastest path to output quality improvement in the MVP.

## Minimum feedback to store
- rating
- output type
- language
- writing level
- whether warning was shown
- comment
- timestamp

## Helpful future feedback fields
- copied output
- downloaded output
- edited after generation
- used in actual application
- interview received later

## Review loop
Weekly review should identify:
- low-rated outputs
- common weak spots
- prompt failure patterns
- UI confusion points
- differences by market, field, and seniority
