# MVP Scope

## Current state

Already working:
- CV generation
- cover letter generation
- candidate profile extraction
- role-fit scoring
- star-rating feedback storage
- DOCX CV upload
- input limits
- tester rollout preparation

## MVP objective

Deliver a robust, credible, and useful application-tailoring assistant for early testers.

## In scope for MVP

### Core candidate flow
- upload CV or paste CV text
- extract structured candidate profile
- create canonical candidate profile
- paste target job description
- show role-fit score and warning
- generate tailored CV
- generate tailored cover letter
- capture output feedback

### Product quality
- improve output quality
- reduce hallucinations
- improve German tone and realism
- basic feedback analytics
- polished tester UI

### Data
- store candidate profile
- store generated outputs
- store ratings and comments
- store generation context such as language and writing level

## Out of scope for this MVP

- automated application sending
- recruiter outreach automation
- interview copilot
- offer negotiation workflow
- full job recommendation engine
- calendar sync
- rejection analytics at scale
- mobile app

## MVP quality bar

The MVP is ready for broader tester rollout when:
- output is stable
- poor-fit warnings are sensible
- generation failures are rare
- feedback collection works
- profile extraction is consistent
- UI is clear enough for non-technical testers
